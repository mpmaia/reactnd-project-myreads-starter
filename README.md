# My Reads

My Reads is an app developed as the first project of the [Udacity's React Nanodegree Course](https://www.udacity.com/course/react-nanodegree--nd019). 

This app can be used to managed reading book lists.

## Installation

Follow this steps to run the App on your machine:

- run `npm install` from the within the above root directory
- run `npm start`
- the app will be running on `http://localhost:3000`
